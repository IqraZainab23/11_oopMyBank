# OOPMyBank

## Overview
This is a simple OOPMyBank Project implemented in Typescript.

## Usage
Run the Project:
```bash
npx 11_oopmybank
```

## Dependencies
`Inquirer`: Provides the interactive prompt interface.

`Chalk`: For styling and coloring the output.
